## 🚀 O que eu estou construindo

*   **[Tenancy Ledger](https://github.com/artcalciolari/TenancyLedger):** Um Monólito Modular estruturado em Bounded Contexts (DDD) com NestJS e PostgreSQL. Focado em garantir a integridade absoluta de dados financeiros e transações de aluguel.
*   **[PDFChat](https://github.com/artcalciolari/PDFChat):** Aplicação de IA local construída com Python, LangChain e TinyLlama, implementando busca vetorial com FAISS e processamento de Speech-to-Text.
*   **[Controle de Estoque](https://github.com/artcalciolari/controle-estoque):** API REST conteinerizada (Node.js/Express + PostgreSQL) com infraestrutura otimizada em Alpine Linux para baixo consumo de recursos via Docker Compose.
*   **[BookFlow](https://github.com/artcalciolari/BookFlow):** Uma biblioteca digital desenvolvida para demonstrar proficiência no ecossistema Frontend (React 19, Vite). Fortemente tipado com TypeScript e focado na integração limpa com APIs REST (Open Library), utilizando custom hooks e documentação automatizada via TypeDoc. Deploy ativo na Vercel.
